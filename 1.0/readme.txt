== Description ==

A Breaking news ticker based on JQuery ,Displays a sliding or fading list of post titles with date and time. The ticker can also be used as a featured posts titles.
Just to give an example you can set the styles according to your needs.

You can display:

    * Today's Posts
    * Featured
    * Categories Wise.
    * Specific Posts
    * Number of Posts to display
    
Admin Can set:

    * Choose Category Filter options
    * Total Posts - N number of posts to display
    * Box styles - Background Color , width and height.
    * Post Styles (Font Color, and Font Size)
    * Title Styles (Text headline , Font color, font size.)
    * Date Styles( Font color and font size)

In sort you can set and do styles sheet for full news ticker.

Support 3 - 3.1.2 (Tested)

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `news-ticker-pluginleopedia` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php showTicker(); ?>` in your templates
